### 1. 🔧 修复跑马灯顿挫问题

**问题**: 跑马灯移动一小段时间后会顿一下，重新调整位置

**原因**: CSS动画在循环时可能有微小的重置

**解决方案**:
- 在动画重新应用时使用 `requestAnimationFrame`
- 在设置动画前强制重排：`void el.offsetWidth`
- 确保动画平滑过渡

```javascript
// 强制重排
void el.offsetWidth

// 使用 requestAnimationFrame 确保平滑
requestAnimationFrame(() => {
  el.style.animation = `${animationName} ${props.speed}s linear infinite`
})
```

### 2. 📐 文字大小改为百分比方式

**之前**: 固定像素值 20-300px  
**现在**: 百分比 10%-100%（基于屏幕高度）

**优势**:
- ✅ 更直观：100% 表示完全铺满屏幕高度
- ✅ 自适应：不同屏幕自动调整
- ✅ 更优雅：按比例缩放

**实现**:
```javascript
// 计算实际字体大小：百分比基于容器高度
const wrapperHeight = wrapper.offsetHeight
const actualFontSize = Math.floor(wrapperHeight * (props.fontSize / 100))
el.style.fontSize = `${actualFontSize}px`
```

**配置界面**:
```
📏 文字大小: 50%
[滑块: 10% ----●---- 100%]
（百分比基于屏幕高度，100%表示完全铺满）
```
