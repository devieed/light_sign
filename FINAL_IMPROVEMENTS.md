- ✅ 支持鼠标点击、触摸、键盘
- ✅ 优雅的淡入淡出动画
- ✅ 配置面板打开时按钮不会隐藏

## 🎯 用户体验流程

### 首次访问
```
1. 打开网站
   ↓
2. 显示欢迎弹窗（文字在背景滚动）
   ↓
3. 点击"开始配置" → 打开设置面板
   ↓
4. 设置文字、颜色等
   ↓
5. 点击"应用设置"
   ↓
6. 设置面板关闭，文字开始滚动
   ↓
7. 5秒后按钮自动隐藏
   ↓
8. 纯净的跑马灯显示（无干扰）
```

### 需要调整设置时
```
1. 点击屏幕任意位置（或触摸/按键）
   ↓
2. 按钮重新出现（淡入动画）
   ↓
3. 点击设置按钮 ⚙️
   ↓
4. 修改设置
   ↓
5. 应用设置
   ↓
6. 5秒后按钮再次隐藏
```

### PC端全屏
```
1. 点击屏幕唤出按钮
   ↓
2. 点击全屏按钮 ⛶ 或按 F11
   ↓
3. 进入全屏模式
   ↓
4. 按钮显示 5 秒后隐藏
   ↓
5. 点击屏幕可再次唤出
   ↓
6. 点击 🗗 或按 ESC 退出全屏
```

### 自动隐藏计时器
```javascript
const startHideButtonsTimer = () => {
  if (hideButtonsTimer.value) {
    clearTimeout(hideButtonsTimer.value)
  }
  
  showButtons.value = true
  
  hideButtonsTimer.value = setTimeout(() => {
    if (!showConfig.value) {  // 配置面板打开时不隐藏
      showButtons.value = false
    }
  }, 5000)  // 5秒后隐藏
}
```
